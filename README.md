# 🚀 Universal Xiaomi Custom Boot Logos Collection

A curated collection of minimalist, clean, and ultra-modern boot logos (splash screens) for various Xiaomi, Redmi, and POCO devices. Built as systemless modules compatible with modern Android root solutions.

---

## ✨ Features
- **Modern Aesthetics:** Minimalist and premium custom boot logos designed to replace boring stock splash screens.
- **Perfect Scaling:** Tailored resolutions matching each specific device's screen properties—no stretching, no pixelation.
- **Systemless & Safe:** No permanent changes to your system partition. Easily reversible.
- **Cross-Platform:** Works flawlessly across different root managers.

## 🛠️ Installation Guide

> ⚠️ **CRITICAL NOTE FOR KernelSU USERS:**  
> If you are using **KernelSU** or **APatch**, you **MUST** have a magic mount back-end installed and active (such as **Magic Mount-rs**, **Hybrid Mount**, or an MKSU environment). Without it, systemless partition overrides for splash files may fail to mount properly. Magisk users can flash directly without any extra steps.

1. Head over to the [Supported Devices](#-supported-devices--downloads) table and download the specific `.zip` module for your device.
2. Open your preferred root manager app (**Magisk**, **KernelSU**).
3. Go to the **Modules** tab.
4. Select **Install from storage** and choose the downloaded `.zip` file.
5. Once the flashing process is successfully completed, **Reboot** your device.
6. Enjoy your fresh, minimalist boot screen!

---

## 🔄 How to Uninstall / Revert to Stock

If you want to restore your device's original boot logo:
1. Open your root manager app.
2. Locate this module in the **Modules** list.
3. Tap **Remove** (or toggle it off).
4. **Reboot** your device, and your stock splash screen will be automatically restored.
